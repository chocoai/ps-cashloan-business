2.3.9 职业相关信息：用户姓名、教育背景、职业经历、职位、职务等信息；

2.3.10 学历学籍信息：学信网用户注册信息、姓名、身份证号码、学历学籍信息（专业、入学/毕业时间、证书照片等）等；

2.3.14 微信相关信息：用户头像、用户昵称等；
