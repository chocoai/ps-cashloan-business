服务密码是中国联通、中国移动、中国电信客户的身份识别密码，由6位数字组合（0～9）。如果忘记了密码，可以选择下面的方式来找回密码：

致电运营商的客服号码，选择人工服务，核实您的身份信息后即可找回密码。

移动客服电话 10086

联通客服电话 10010

电信客服电话 10000

2. 打开电脑浏览器，访问运营商的网站，您可以在运营商的官网上重置密码。

移动的官方网站：http://www.10086.cn/

联通的官方网站：http://www.10010.com/

电信的官方网站：http://www.189.cn/

3. 请携带您的身份证等证件，前往各地的运营商营业厅柜台办理。

无法收到验证码？

如果您无法收到手机短信验证码，请尝试下面的方法解决：

1. 检查您的手机是否暂无信号或者已经欠费

2. 确认您的手机上是否安装了安全软件，导致短信验证码被拦截或屏蔽

3. 访问您手机运营商的官方网站，并尝试登录营业厅，确认您的手机是否因为验证码发送过于频繁导致收不到验证码。各手机运营商官方网站为：

移动 http://www.10086.cn/

联通 http://www.10010.com/

电信 http://www.189.cn/

如果以上方法都没有解决，可能是暂时的网络延迟，您可以稍后尝试重新获取。

授权过程需要多久？

根据运营商网络情况不同，授权服务需要1-3分钟时间才能完成，请耐心等待。 如果您发现等待时间过长，请选择非网络高峰期的时间重新授权。
